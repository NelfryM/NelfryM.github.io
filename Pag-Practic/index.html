HTML

Es el lenguaje estándar utilizado para crear y diseñar páginas web. Nos permite definir la estructura de la página y el contenido que se va a mostrar en el navegador. Los archivos HTML tienen extensión .html


CSS

Es el lenguaje utilizado para modificar el diseño y presentación de las páginas web. Cambia su estilo y apariencia incluyendo cosas como: colores, fuentes, márgenes, tamaños de texto y muchas otras más.

Se les denomina hojas de estilo «en cascada» porque puedes tener varias y una de ellas con las propiedades heredadas (o «en cascada») de otras.

Elementos HTML

Elementos Comunes de HTML(Lenguaje de Marcado de Hipertexto), es la piedra angular de la creación de páginas web. Entre sus elementos más utilizados se encuentran:


  • Etiquetas de Encabezado (h1 a h6):

    Definen los títulos y subtítulos, siendo h1 el más importante.


  • Párrafo (p):

    Se utiliza para delimitar bloques de texto.


  • Enlaces (a):

    Permiten la navegación entre páginas mediante hipervínculos.


  • Imágenes (img):

    Incrustan imágenes en la página, utilizando atributos como src para la dirección de la imagen y alt para la descripción. Estos elementos son fundamentales para estructurar el contenido y facilitar una experiencia de usuario coherente y accesible en la web. Además, se complementan con CSS para mejorar la presentación visual y con JavaScript para añadir interactividad.


Elementos CSS

CSS, o Hojas de Estilo en Cascada, es esencial para el diseño web moderno. Entre sus elementos más utilizados están los selectores, que identifican elementos HTML para aplicarles estilos. Las propiedades como color, font-size, margin y padding definen el aspecto visual de los elementos. Las unidades de medida, como px, em, rem, y %, permiten un diseño flexible y adaptable a diferentes dispositivos. Además, las media queries son cruciales para el diseño responsivo, ajustando estilos según el tamaño de la pantalla. Finalmente, Flexbox y Grid son herramientas poderosas para crear diseños complejos y responsivos, facilitando la alineación y distribución de elementos en una página web. Estos elementos de CSS son fundamentales para crear interfaces atractivas y funcionales.

Grid con elementos

Margin

El margen define el espacio exterior alrededor de un elemento.

Padding

El relleno define el espacio interior entre el contenido de un elemento y su borde.

Border

La propiedad borde define el borde alrededor de un elemento, incluyendo su estilo, ancho y color.

Background

La propiedad de fondo permite definir el color, imagen, posición y repetición del fondo de un elemento.

Flexbox

Flexbox es un modelo de diseño unidimensional que se utiliza para crear layouts flexibles.

Grid

CSS Grid Layout es un sistema de diseño bidimensional que permite crear layouts complejos con facilidad.

Font

Las propiedades de fuente permiten definir la familia, tamaño, peso, estilo y otras características de las fuentes de texto.

Color

La propiedad de color define el color del texto de un elemento.
